# CSE310---BookStore--Module1

# Overview

This application is part of the activities for the CSE-310 course, being a basic library application with the ability to search a list of books, and allow the borrowing and return of books.

It is an ASP.NET application using Blazor.

# Software Description
The software is a basic Library application using C#, ASP Net and Blazor.

# Purpose of creating this software
The goal is to practice basic application development skills by building an application that interacts with a relational database. This allows other developers to see and learn the basic structure of a database-integrated solution in C#.
